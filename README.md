## 簡介 📖

此作品為 Vue.js 前端練習，主要功能有以下：

  1. 檢視 [全部＆未完成＆完成] 待辦事項。

  2. 可輸入預計要做的待辦事項。

  3. 可刪除已在清單內的任一待辦事項。

## 使用技術＆練習技術 🚀

- Vue CLI 3
- Vuex
- Vue Router
- Vue Components
- JavaScript (ES6)
- ESLint (Standard)
- Web Storage (Local Storage)
- Google Analytics
- CDN Service (Cloudflare)
